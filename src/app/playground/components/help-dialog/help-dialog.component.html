O que é o CLI Playground?

O CLI Playground é uma ferramenta que permite que você teste comandos do Azure CLI sem precisar instalar nada em sua máquina. Você pode testar comandos do Azure CLI e ver o que eles fazem.

Os comandos são executados em uma API que simula o Azure CLI, então você pode testar comandos sem se preocupar em criar recursos reais no Azure.

Esta ainda é uma feature em construção. Para mais informações sobre a API do CLI, acesse o repositório GitHub.
